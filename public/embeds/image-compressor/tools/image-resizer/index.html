Frequently Asked Questions

Common questions about resizing images online.

Does resizing reduce image quality?

Downscaling (making an image smaller) preserves quality well. Upscaling (making it larger) will soften the image because there is no new detail to add — the browser interpolates between existing pixels. Always resize down from a high-resolution source for best results.

What does "lock aspect ratio" mean?

Aspect ratio is the proportional relationship between width and height. Locking it means that when you change one dimension, the other updates automatically to keep the image's original proportions. Unlocking lets you set width and height independently, which will stretch or squash the image.

What is the difference between px and % resize?

Pixel (px) resize sets exact output dimensions. Percentage (%) resize scales relative to the original — for example, 50% of a 2000 × 1500 image produces a 1000 × 750 image. Percentage is useful when you want to uniformly scale down without calculating exact pixel values.

What size should I use for a website hero image?

For a full-width desktop hero, 1920 × 1080 px (Full HD) is the standard. If the image is not full-bleed, 1200 × 630 px is a good balance of quality and file size. Always compress after resizing to keep load times fast.

Can I resize to a specific file size instead of pixel dimensions?

Pixel dimensions and file size are related but not the same thing — file size also depends on compression and color complexity. To hit a target file size, use the Image Compressor tool after resizing, and adjust the quality slider until you reach your desired size.

Is there a maximum image size this tool can handle?

The tool processes images in your browser using the Canvas API, which supports up to 16,000 × 16,000 px in most modern browsers. Very large images (above 4000 px on either side) may be slow to process on lower-powered devices.
