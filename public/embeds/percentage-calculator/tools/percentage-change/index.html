How to Calculate Percentage Change

Percentage change tells you how much a value has grown or shrunk compared to where it started. The formula always divides by the original value — this is what makes it a relative comparison rather than an absolute one.

The Formula

% Change = ((New Value − Old Value) ÷ |Old Value|) × 100

The vertical bars around Old Value denote the absolute value. Dividing by a negative original would flip the sign unexpectedly, so the absolute value keeps the direction unambiguous.

Step-by-Step Example: Price change from $80 to $100

  1. Find the change: $100 − $80 = $20
  2. Divide by the original: $20 ÷ $80 = 0.25
  3. Multiply by 100: 0.25 × 100 = 25%
  4. Answer: 25% increase

Percentage Change vs. Percentage Difference

These two concepts are often confused but answer different questions:

Percentage ChangePercentage Difference
Question answeredHow much did X change from before to after?How different are X and Y from each other?
Reference pointThe original (old) valueThe average of both values
Directional?Yes — result can be positive or negativeNo — always a positive distance
Best used forBefore/after comparisons (price, salary, weight)Comparing two equal peers (two prices, two readings)
Formula denominator|Old Value|(Value₁ + Value₂) ÷ 2

Common Mistakes

  • Swapping old and new. The order matters — a price rising from $100 to $120 is a 20% increase, but the same values reversed (from $120 to $100) is a 16.7% decrease.
  • Confusing absolute change with percentage change. A salary going from $30,000 to $33,000 increased by $3,000 in absolute terms, but by 10% in relative terms. The percentage is usually more meaningful for comparisons.
  • Starting from zero. If the original value is 0, percentage change is undefined — you cannot divide by zero. Use absolute change instead in that case.

Frequently Asked Questions

What is percentage change?▶
Percentage change measures how much a value has increased or decreased relative to its original value, expressed as a percentage. A positive result means an increase; a negative result means a decrease.
What is the formula for percentage change?▶
Percentage Change = ((New Value − Old Value) ÷ |Old Value|) × 100. You divide by the absolute value of the original to ensure the sign of the result reflects the direction of change.
How do I calculate a percentage increase?▶
Subtract the original value from the new value to get the absolute change, divide by the original value, then multiply by 100. For example, from 80 to 100: (100 − 80) ÷ 80 × 100 = 25% increase.
How do I calculate a percentage decrease?▶
Use the same formula: ((New − Old) ÷ |Old|) × 100. When the new value is smaller, the result will be negative. For example, from 200 to 150: (150 − 200) ÷ 200 × 100 = −25% (a 25% decrease).
What is the difference between percentage change and percentage difference?▶
Percentage change has a clear direction — it compares a new value to a defined original. Percentage difference is symmetric — it compares two values when neither is the reference, using their average as the denominator. Use percentage change for before/after comparisons; use percentage difference when the two values are equally valid starting points.
Can percentage change be more than 100%?▶
Yes. If a value doubles, that is a 100% increase. If it triples, that is a 200% increase. There is no upper limit on percentage increase. Percentage decrease, however, is capped at −100% (the value reaching zero).
What is the percentage change from 50 to 75?▶
(75 − 50) ÷ 50 × 100 = 50% increase. The value grew by 25, which is half (50%) of the original value of 50.
What does a 25% decrease from 200 give you?▶
A 25% decrease from 200 gives 150. To calculate: 200 × (1 − 0.25) = 200 × 0.75 = 150. Alternatively, find 25% of 200 (= 50) and subtract: 200 − 50 = 150.
